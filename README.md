# FreshCycle

## AI-Based Smart Platform for Fruits and Vegetables Recycling and Redistribution

FreshCycle is a Flutter-based mobile application developed to reduce food wastage in supermarkets and local markets by enabling smart recycling and redistribution of fruits and vegetables.

The application helps vendors and users identify reusable food items, manage waste efficiently, and support sustainable food distribution using AI-based solutions.

---

# Features

- User Login and Registration
- Firebase Authentication
- Fruits and Vegetables Upload
- Waste Detection and Classification
- Redistribution Management
- Donation Support System
- Smart Food Waste Handling
- Real-Time Data Management
- Clean and User-Friendly Interface

---

# Technologies Used

- Flutter
- Dart
- Firebase
- Cloud Firestore
- Firebase Authentication
- Android Studio

# Project Objectives

- Promote sustainable recycling

- Improve waste management systems
